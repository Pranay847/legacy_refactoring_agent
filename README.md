# Legacy Refactoring Agent

An AI-powered tool that analyzes and refactors legacy codebases, helping developers modernize old code automatically.

## Project Structure

```
legacy_refactoring_agent/
├── backend/    # Python server & agent logic
└── frontend/   # JavaScript UI
```

## Getting Started

### Prerequisites
- Python 3.8+
- Node.js & npm

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Features

- Analyzes legacy code and identifies areas for improvement
- Suggests and applies refactoring transformations
- Clean UI to upload and review refactored code

## Tech Stack

- **Backend:** Python
- **Frontend:** JavaScript
